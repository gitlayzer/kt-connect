Downloads And Install
---

<!-- tabs:start -->

#### ** MacOS **

Manually download the latest package:

* [MacOS Intel](https://github.com/gitlayzer/kt-connect/releases/download/v0.3.7/kt-connect_0.3.7_darwin_amd64.tar.gz)
* [MacOS Apple Silicon](https://github.com/gitlayzer/kt-connect/releases/download/v0.3.7/kt-connect_0.3.7_darwin_arm64.tar.gz)

Downloads & install (use Intel edition as example):

```
$ curl -OL https://github.com/gitlayzer/kt-connect/releases/download/v0.3.7/kt-connect_0.3.7_darwin_amd64.tar.gz
$ tar zxf kt-connect_0.3.7_darwin_amd64.tar.gz
$ mv ktctl /usr/local/bin/ktctl
$ ktctl --version
```

#### ** Linux **
