下载和安装
---

<!-- tabs:start -->

#### ** MacOS **

手工下载最新软件包:

* [MacOS x86 64位](https://github.com/gitlayzer/kt-connect/releases/download/v0.3.7/kt-connect_0.3.7_darwin_amd64.tar.gz)
* [MacOS ARM 64位](https://github.com/gitlayzer/kt-connect/releases/download/v0.3.7/kt-connect_0.3.7_darwin_arm64.tar.gz)

通过命令行下载并安装KT（以x86 64位版本为例）

```bash
$ curl -OL https://github.com/gitlayzer/kt-connect/releases/download/v0.3.7/kt-connect_0.3.7_darwin_amd64.tar.gz
$ tar zxf kt-connect_0.3.7_darwin_amd64.tar.gz
$ mv ktctl /usr/local/bin/ktctl
$ ktctl --version
```
